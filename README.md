# 🚀 LeetCode Solutions

> Siêu đơn giản: Tạo file → Code → Push GitHub

## ⚡ Cách dùng

```bash
make new    # Tạo file problem mới
# [Code solution của bạn]
make push   # Đẩy lên GitHub
```

**Xong!**

## 📝 Ví dụ

```bash
make new
# Problem number: 1
# Problem name: TwoSum
# ✅ Created: problems/Problem1.kt

# [Bạn code trong file Problem1.kt]

make push  
# Commit message: Two Sum solution
# 🎉 Pushed to GitHub!
```
